# 앱 아이콘 및 스플래시

Expo 실행 전에 다음 이미지를 이 폴더에 추가하세요.

- **icon.png** – 1024x1024 (앱 아이콘)
- **splash-icon.png** – 스플래시 화면용
- **adaptive-icon.png** – 1024x1024 (Android 적응형 아이콘)

새 Expo 프로젝트에서 복사해 오려면:

```bash
npx create-expo-app@latest _tmp --template blank-typescript
# Windows: xcopy _tmp\assets .\assets /E /I
# macOS/Linux: cp -r _tmp/assets/* ./assets/
# 이후 _tmp 폴더 삭제
```
